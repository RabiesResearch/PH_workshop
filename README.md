# Workshop: Rabies in MIMAROPA

Following data provided:
- `data/epi/dat_all_anon.csv`: Line list of anonymised confirmed and probable cases and human deaths
- `data/epi/mdv_summary.csv`: Summary of mass dog vaccination events in MIMAROPA
- `data/gis`: Contains shape files (`PH_municipality`) and village centroids (`PHL_village_centroids.csv`) for mapping
